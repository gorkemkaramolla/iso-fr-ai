## Quick start

**Clone the Project**
```bash
mkdir isoai
cd isoai
git clone https://github.com/gorkemkaramolla/iso-fr-ai.git .
code .
```

**Run Backend Code**
```bash
python -m venv .
.\Scripts\activate
pip install -r requirements
cd insightface
python ip_cam_backend.py
```

**Open Local Camera**
```bash
.\Scripts\activate
cd insightface
python client_camera.py
```

**Run whole application futures**
```bash
.\Scripts\activate
cd flask
python app.py
```

**Run Frontend Code**
```bash
cd iso-electron
npm i
npm run dev
```
